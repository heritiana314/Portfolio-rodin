COUVEUSE

DESCRIPTION

   This project involves the design and development of a smart, multi-functional incubator system intended for domestic use. This innovative device was created to address various needs, including the incubation of different types of eggs, yogurt fermentation, and fruit ripening. Its adaptable design ensures precise control of environmental parameters (temperature, humidity, oxygen, etc.), tailored to each specific application.

   To facilitate control and monitoring, the incubator is connected to a dedicated web platform named "Couveuse." This interface allows users to monitor and operate the device remotely in real-time, providing a simple and accessible management solution from anywhere.

   This project highlights skills in biology, mechanics, electronics design, automation, thermodynamics, web development, and IoT integration , combining hardware and software to create a modern and practical solution tailored to users' needs.
